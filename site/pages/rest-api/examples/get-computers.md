---
title: Get Computers
---

Retrieves the list of computers registered in the console.

# cURL

```
curl -X GET --user {username}:{password} https://{server}:{port}/api/computers
```

# python

```python
import requests
r = requests.get('https://{server}:{port}/api/computers', auth=('{username}', '{password}'))
print(r.text)
```
