---
title: Get Analyses
---

Get a site's available analyses.

# cURL

```bash
curl --user {username}:{password} https://{server}:{port}/api/analyses/{site}
```

# python

```python
import requests
r = requests.get('https://{server}:{port}/api/analyses/{site}', auth=('{username}', '{password}'))
print(r.text)
```
